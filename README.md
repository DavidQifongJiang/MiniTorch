# MiniTorch
MiniTorch is a PyTorch-style deep learning framework built from scratch in Python to understand and reimplement the core layers of modern ML systems: automatic differentiation, tensor abstractions, optimized CPU kernels, CUDA acceleration, and neural-network training.


Module 0 — Foundations: core mathematical operators and higher-order functional primitives

Module 1 — Scalar Autodiff: computation graphs, reverse-mode autodiff, central-difference gradient checks

Module 2 — Tensor Engine: multidimensional tensors, broadcasting, reductions, shape transforms, tensor autodiff

Module 3 — Performance Backends: Numba-optimized CPU kernels and CUDA kernels for map/zip/reduce/matmul

Module 4 — NN Primitives: pooling, softmax, dropout, and end-to-end training on nonlinear tasks
